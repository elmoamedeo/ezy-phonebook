# How to run the project

## Run the project

```bash
./gradlew bootRun
```

## Run the tests

```bash
./gradlew test
```

## Run the tests with coverage

```bash
./gradlew jacocoTestReport
```

## Run the tests with coverage and open the report

```bash
./gradlew jacocoTestReport && open build/reports/jacoco/test/html/index.html
```
